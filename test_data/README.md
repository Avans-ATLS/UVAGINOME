This subdirectory contains 2 small datasets for UVAGINOME pipeline testing.

# Data
Both files are `.fastq.gz` files containing randomly subsampled 16S rRNA gene ONT sequencing data.
Each sample contains 5000 reads.

# Usage
Even though both files contain 16S data, they can still be used to test the metagenomic part of the pipeline.

To use the supplied test data, create a samplesheet for it with the `samplesheeter.py` script in this repository.
